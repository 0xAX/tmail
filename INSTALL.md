Build and installaion instructions
==================================

Clone repo:

```
$ git clone git@github.com:0xAX/tmail.git
```

Go to clonned repo:

```
$ cd tmail
```

and execute make:

```
$ make
```

or:

```
$ gmake
```

on `*BSD` systems.
